자동자 경주 요구사항

- 주어진 횟수 동안 n대의 자동차는 전진 또는 멈춤
  - 몇번 움직일지에 대한 입력값과 n대의 자동차의 수를 입력 받아야함
    - 입력값들에 대한 유효성 검증 
  - 각 횟수 마다 이동 횟수를 출력 
    - 이동 조건은 0~9 랜덤값을 구한 후 4이상일 경우
      - 0~9사이의 랜덤값 구하기
      - 랜덤값이 4이상일 경우에만 출력
- 구현 순서
  1. 입력값 받기
  2. 랜덤값 구하기
  3. 유요한 랜덤값

      
- 프로그래밍 요구 사항
  - 모든 로직에 단위 테스트 구현할. UI 로직 제외
    - 핵심 로직과 UI로직 구분
    - UI로직을 입, 출력에대한 클래스를 추가해 분리
  - 자바 코드 컨벤션에 따라 코딩
  - else 예약어 금지
